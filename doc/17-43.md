# Build dockerized golang program

```Dockerfile
FROM scratch
COPY $app_id /
ENTRYPOINT ["/$app_id"]
EOM
```
```bash
docker run --rm \
          -e CGO_ENABLED=true \
          -e COMPRESS_BINARY=true \
          -v "$(pwd -P):/src" \
          -v /var/run/docker.sock:/var/run/docker.sock \
          centurylink/golang-builder \
          bvberkum/x-ci:hello-world
```
